1: clone Directory
2: navigate to your cloned directory (example: "D:\we15-lab4")
3: start activator with command "activator" (check your PATH settings)
4: create intelliJ idea files with command "idea"
5: exit activator with command "exit"
6: (optional) run activator with command "activator ~run"
7: import directory with intelliJ. you will get the following message:

"The project 'we15-lab4' has an older format and will be converted. You may not be able to open the project with earlier versions of IntelliJ IDEA. Details...  Old versions of project files will be saved to: 'D:\TU Wien\Web Engineering\UE4-Angabe\we15-lab4\projectFilesBackup'"

8: click CONVERT
9: intelliJ will ask you to add files to the repo, click NO.
10: delete the Folder "projectFilesBackup"
11: git status must now be clean.
